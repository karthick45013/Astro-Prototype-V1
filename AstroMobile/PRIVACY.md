# Astro Mobile Privacy Notes

Astro Mobile is designed as a companion app for Astro Desktop.

## Data Sent To Desktop

- The command text you type in the mobile app.
- The pairing token included in the desktop QR/link.

## Data Stored On Phone

- The paired desktop URL is saved locally so you do not need to paste or scan it every time.

## Camera Permission

Camera access is used only to scan the Astro Desktop pairing QR code.

## Network

Astro Mobile connects directly to Astro Desktop using the URL shown by the desktop app. It does not need a cloud account for local hotspot/Wi-Fi use.

## Important

Only pair with a desktop you trust. Anyone with the pairing URL and token may be able to send commands to that running Astro Desktop session.
